Seven Days of Linux A blog post by Don Kuntz, with 3 comments

This is my seventh day of linux as my primary operating system.

So far, so good. My notable problems have all been solved (last night too, I upgraded from 32bit Ubuntu to 64bit, now everything is almost exactly the same on both my laptop and my desktop.

So, let’s start with what I’ve got now compared to the initial “Praxis’ Software” list:

  • Rhythmbox be gone! Having, last night, upgraded to 64bit, everything is now all peachy between Praxis and Banshee.
  • I like Gwibber. I like it more than #newtwitter, and it comes preinstalled, bonus.
  • I downloaded Filezilla, I forgot to grab an ftp client (oops)

As for things I’ve been feeling like mentioning but haven’t:

  • I like having a universal menu. Sure it’s a bit different than what I’m used to, but it keeps windows less cluttered and gives everything a more unified interface. There are still some apps that don’t support it (and never will – chrome), but I’m okay with that.
  • With the universal menu, I’ve slimmed from the menu bar to the main menu (which is just the ubuntu logo with a drop down). Less space taken up in the upper panel, giving more room to the universal menu. It also fits in better with the universal menu .

Also, moving between installs, I’ve learned that I really like having a script which will do all the setup stuff for me. I only needed to do three things after running my setup script, add in and integrate the universal menu, fix my own menus (I don’t use the default ones, I’ve instead divided things up into Internet, Create, Play, Media and Tools. Same for my laptop), and set the icon set to Faenza (I’m pretty sure I could have done two of the three with my script but didn’t really want to look into it when I was writing the script)

Alright, now that that’s out of the way, I love the terminal. Having used Windows as my primary operating system I never really had to use the command prompt, but when I did it always felt really clunky. That’s not the case with the terminal. I just can’t say enough about it, so I’ll just stop.

My search for a more Windows 7 like taskbar is still in vain, I’m still using Docky, but at least it’s in panel mode.
