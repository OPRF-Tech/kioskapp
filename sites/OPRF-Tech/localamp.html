A Local *AMP Server

LAMP - Linux
Most Linux distros use either apt-get/aptitude or yum. Ubuntu, the distro we use uses apt-get. This guide is written for an apt-get/aptitude distro (if you’re using an aptitude distro, replace all instances of apt-get with aptitude
  • Open your terminal and type the following:
    sudo apt-get install apache2 php5 php5-gd mysql-server mysql-client php5-mysql libapache2-mod-php5 php5-curl php5-xmlrpc php5-pear
  • Bravo, you’ve now got a LAMP server installed. Your www folder (the directory that contains your web files) is in /var/www, I suggest running a second terminal command that unlocks that directory perminantly
    sudo chmod -r 0777 /var/www
MAMP - Mac
You need to be running 10.4.0 or higher to use MAMP
MAMP does not load on login
  • Head over to http://mamp.info, and download just the standard mamp (not mamp pro).
  • Drag the MAMP file out of the dmg and put it in your applications folder.
  • Start MAMP, Start Apache and MySQL
  • Open the Preferences, choose the Ports tab, and change the Apache port to 80 (this is the default port looked at when visiting 127.0.0.1)
  • Now open the Apache tab, change the Document root to something else (unless you like where it is). I suggest /users/[name]/www (or htdocs instead of www).
WAMP - Windows
There are lots of options for windows, however EasyPHP, the one shown here is the easiest to setup and use. You can also use EasyPHP on a flash drive.
EasyPHP does not load on login - you can tell it to do so if you want through the control panel
  • Head over to http://www.easyphp.org and download EasyPHP (not a module pack).
  • Run the installer when it’s done downloading.
  • Choose a directory you want for it to be installed in (I use c:\users\[name]\localhost. If you want to do that, after you choose the localhost folder (which you have to make) remove the \easyphp[version] from the install location string)
  • I personally change the start menu text from easyphp[version] to just easyphp, but you can choose whatever you want.
  • Now head over to your chosen directory and run EasyPHP[version].exe
    • If you go to 127.0.0.1 and don’t see anything or get an error, open easyphpdir\conf_files and open httpd.conf in your text editor
    • Now search for 127.0.0.1. If you find any instances that aren’t commented out (that don’t have a # before them) and aren’t 127.0.0.1:80, replace them to be 127.0.0.1:80
    • Now try going to 127.0.0.1 in your browser again

FAQ

  1. What is *AMP?
    *AMP is * (variable placeholder) Apache MySQL PHP server. The variable placeholder is for your computer’s operating system, L for Linux (LAMP), M for Mac (MAMP) and W for Windows (WAMP)
  2. Why do I want One?
    An *AMP server is useful for local development of php/mysql or higher level web development projects. I essentially gives you a server to play with that’s only accessible on your computer, so it’s just for you.
  3. How will I know it's working?
    Open a web browser and head over to 127.0.0.1, if you get any sort of It Works page, you’re golden (though some may have different pages shown)
